EcoGuard

EcoGuard is an AI-powered solution designed to detect illegal deforestation and poaching in real-time. It employs a VGG-16 model trained on the ESC-50 dataset for audio recognition of machinery, enhancing detection accuracy. The system integrates OpenCV for image processing and Librosa for audio analysis, enabling efficient monitoring and alerting authorities about potential illegal activities.

Roomie

Roomie is a hostel and roommate-finding app designed to simplify the search process for university students. It features a Django-based backend, with a frontend built using HTML and CSS, ensuring a seamless user experience. The app uses SQLite for data storage, enabling efficient query handling and making it easier for students to find suitable accommodation and roommates.

Bouddhanath Stupa

Capturing the essence of Boudhanath Stupa in a stunning 3D model, this project utilizes C++ and OpenGL to create a meticulously crafted 3D model. With realistic rendering and smooth interactions, it offers an immersive visual experience, blending art and technology to showcase one of the world’s most revered landmarks.

CourseSelect

CourseSelect is an elective course registration system designed for BCT students at IOE Campus as part of a DBMS project. Built with MySQL, the system enables efficient course selection, enrollment, and management. It simplifies the registration process, ensuring smooth and organized handling of elective courses for students.
